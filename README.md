# Sokoban-C

### 简介

一款简陋的C语言控制台推箱子小游戏

### 开发环境

Windows系统下的DevC++

### 使用说明

1. 将项目克隆或下载到自己本地

2. 使用DevC++打开项目目录下的sokoban.dev文件

3. 使用DevC++编译运行

4. 使用英文输入法下的WASD键移动，键盘上的上下左右键无法移动
